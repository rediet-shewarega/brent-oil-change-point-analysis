## Interactive Dashboard

The Task 3 dashboard provides an interactive interface for exploring Brent
oil prices, Bayesian change-point results, volatility indicators, and major
market events.

### Architecture

- Flask backend for data APIs
- React and Vite frontend
- Recharts for interactive visualizations
- CSV and JSON model outputs generated by the Task 2 notebook

### Dashboard features

- Historical Brent oil price chart
- Daily, weekly, and monthly aggregation
- Date-range filters
- Bayesian change-point marker
- Major event markers
- Event selection and drill-down
- Average price and volatility indicators
- Responsive desktop, tablet, and mobile layouts

### Backend setup

From the repository root:

```bash
source .venv/bin/activate
pip install -r dashboard/backend/requirements.txt
cd dashboard/backend
